# 🚀 Alien Galaxy Escape: Gravity Well Edition

A space-themed browser game where you pilot an alien spaceship through the solar system, dodging obstacles while navigating real planetary gravity wells.

## 🎮 How to Play

- **Space / Click / Tap** — Fire ion thrusters (jump)
- Avoid obstacles: asteroid walls, plasma storms, energy fields
- Each planet exerts real gravitational pull — adjust your flight path!
- Score increases with every obstacle cleared
- Speed increases every 8 obstacles

## 🌍 Features

- 10 Solar System planets with unique gravity & visuals
- 4 obstacle types (asteroid walls, debris fields, plasma storms, wormholes)
- Particle explosion effects
- Combo flash alerts
- High score saved locally
- Fully responsive (desktop + mobile)

## 🚀 Run Locally

Just open `index.html` in any browser — no server needed.

```bash
open index.html
```

## 🌐 Deploy on GitHub Pages

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to `main` branch → `/ (root)`
4. Save — your game will be live at:

```
https://Spideralok6.github.io/AlienGalaxyEscape/
```
